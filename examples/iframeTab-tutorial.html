iframeTab 文檔

引入 iframeTab

基於本插件是以Gentelella為框架開發的，樣式對此框架有依賴性，但插件實際使用上對樣式沒有任何依賴性，因此不建議引用css文件。

<!DOCTYPE html>
<html>
<header>
    <meta charset="utf-8">
    <!-- 引入 jQuery 和 iframeTab 文件 -->
    <script src="jquery.min.js"></script>
    <script src="iframeTab.min.js"></script>
</header>
</html>

鏈接Dom

<a href="iframeTab-demo.html" data-num="0">主頁</a>

必須包含 data-num="0"，以此區別一般瀏覽器新開窗口與以iframeTab開啟的鏈接
data-name 為可選屬性，默認a標籤內文字，優先級：data-name > a.text()
data-mul 為可選屬性，若存在，則此鏈接可同時多個創建窗口
data-reload 為可選屬性，若存在，則再次點擊此鏈接時，iframe將被重新加載

標籤欄結構

<ul>
    <li data-tab="iframeTab-demo">標籤一 <i data-btn="close"></i></li>
    <li data-tab="">標籤二 <i data-btn="close"></i></li>
</ul>

HTML標籤可以隨意更換，只要確保 'tabLi' 確實包含在 tabUl 之下，包含 data-btn="close" 的元素於 tabLi 下，之間添加別的元素也是沒有關係的。

標籤欄結構

同上iframe結構只要層級正確，之間添加什麼都可以。

<div>
    <div>
        <iframe src="iframeTab-demo.html" data-iframe="iframeTab-demo" data-num="0" marginheight="0" marginwidth="0" frameborder="0" scrolling="no" onload="iframeTab.iFrameHeight()"></iframe>
    </div>
</div>

第二個div為 tabPan，用來存放 <iframe>，第一個為 tabBody，以存放 tabPan。

初始化 iframeTab

iframeTab.init();

通過 iframeTab.init 方法即可初始化iframeTab，但我相信你會想自行配置一下。

var iframeTabOption = {
    tabUl: '.tabs-header ul',
    tabLi: '.tabs-header li',
}

iframeTab.init(iframeTabOption);

配置項

  • tabUl | String

    標籤頁模塊，[ default: '.tabs-header ul' ]

  • tabLi | String

    標籤頁元素，[ default: '.tabs-header li' ]

  • tabBody | String

    內容容器模塊，[ default: '.tabs-body' ]

  • tabPan | String

    內容容器，[ default: '.tab-panel' ]

  • closesBtnClass | String

    刪除按鈕樣式，[ default: 'fa fa-close' ]

  • newTabPan | String

    包裹iframe的容器，若tabPan有修改，此處應跟隨修改，[ default: '<div class="tab-panel active"><div class="right_col" role="main"></div></div>' ]

API

iframeTab.iFrameHeight()

刷新iframe窗口高度。 在某些場合上如通過JS對頁面進行動態修改，頁面高度改變會出現窗口過長或過短，此時可通過此方法刷新頁面高度。

gulp

因為本插件涉及iframe操作，在本地直接打開將報錯，你需要在服務器上調試，本插件提供 gulp 調試方式，你可以以此進行插件模擬和代碼編譯。

  • gulp 運行服務器并編譯所有代碼
  • gulp browser-sync 運行服務器
  • gulp watch 進入watch模式
  • gulp sass-to-css 將sass編譯成css
  • gulp minify-css 壓縮css
  • gulp jscompress 壓縮js
