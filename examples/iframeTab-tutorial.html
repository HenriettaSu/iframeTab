iframeTab 文檔

引入 iframeTab

本插件是以Gentelella為框架開發的，本demo對此框架有一定依賴性，但實際使用上并非必要。

<!DOCTYPE html>
<html>
<header>
    <meta charset="utf-8">

    <!-- 引入 iframeTab 樣式文件，非必要，可自行定制 -->
    <link href="iframeTab.min.css" rel="stylesheet">

    <!-- 引入 jQuery 和 iframeTab 文件 -->
    <script src="jquery.min.js"></script>
    <script src="iframeTab.min.js"></script>
</header>
</html>

鏈接Dom

<a href="iframeTab-demo.html" data-num="0">主頁</a>

必須包含 data-num="0"，以此區別一般瀏覽器新開窗口與以iframeTab開啟的鏈接
data-name 為可選屬性，默認a標籤內文字，優先級：data-name > a.text()
data-mul 為可選屬性，若存在，則此鏈接可同時多個創建窗口
data-reload 為可選屬性，若存在，則再次點擊此鏈接時，iframe將被重新加載

標籤欄結構

<ul>
    <li class="active tab-keep" data-tab="my-desktop.html" data-num="0">首頁</li>
    <li data-tab="tabPage.html">標籤頁 <i data-btn="close"></i></li>
</ul>

<li> 中的 data-tab 應與對應的iframe頁 <iframe> 中的 src 和 data-iframe 保持一致，內容應為iframe的 src 屬性。同時也必須包含 data-num="0" 。
li.active 表示當前標籤已激活，li.tab-keep 表示右鍵刪除所有標籤時將保留當前標籤。
若需添加 data-btn="close" 為刪除標籤按鈕。

iframe結構

<div class="tabs-body">
    <div class="tab-panel tab-keep active">
        <!-- 包裹iFrame的外部元素，可按自己需求更改，如需設置，應同時在option中配置iframeBox -->
        <div class="right_col" role="main">
            <iframe src="my-desktop.html" data-iframe="my-desktop.html" data-num="0" marginheight="0" marginwidth="0" frameborder="0" scrolling="no" onload="iframeTab.iframeHeight()" height="188"></iframe>
        </div>
    </div>
</div>

<iframe> 中的 src 和 data-iframe 應與對應的標籤 <li> 中的 data-tab 保持一致，內容應為iframe的 src 屬性。同時也必須包含 data-num="0" 。
.tab-panel.active 表示當前iframe已激活，.tab-panel.tab-keep 表示右鍵刪除所有標籤時將保留當前iframe頁。
<div class="right_col" role="main"> 為 div.tab-panel 和 <iframe> 間的容器，並非必要，若需自行配置或不需要該容器，可在option中配置 iframeBox 參數。

初始化 iframeTab

iframeTab.init();

通過 iframeTab.init 方法即可初始化iframeTab，可根據需求修改配置數據。

var iframeTabOption = {
    closesBtnClass: 'fa fa-close',
    callback: {
        onChange: function () {
            alert('標籤切換了')
        }
    }
}

iframeTab.init(iframeTabOption);

配置項

  • tabLiClass | String

    標籤欄標籤 <li> 樣式，[ default: '' ]

  • tabPanClass | String

    標籤頁外層容器 <div class="tab-panel"> 樣式，[ default: '' ]

  • closesBtnClass | String

    刪除按鈕 [data-btn="close"] 樣式，[ default: 'fa fa-close' ]

  • switchBtnDown | String

    折疊按鈕折疊狀態樣式，[ default: 'fa fa-chevron-down' ]

  • switchBtnUp | String

    折疊按鈕展開狀態樣式，[ default: 'fa fa-chevron-up' ]

  • contextmenuClass | String

    右鍵菜單 div.tab-contextmenu 樣式，[ default: 'dropdown open' ]

  • iframeBox | String

    div.tab-panel 和 <iframe> 間的容器，非必要，可配置為 '' ，若Html中有修改，此處應跟隨修改，[ default: '<div class="right_col" role="main"></div>' ]

  • singleLineheight | Number

    標籤欄單行狀態高度，[ default: 92 ]

  • menuWidth | Number

    某些UI框架下，如本demo，菜單欄位於右側，iFrame內容頁並非佔據整個視窗寬度，此情況下應配置菜單欄寬度，[ default: 230 ]

  • callback.onCreat | Function ()

    用於標籤創建時的事件回調函數，[ default: null ]

  • callback.onChange | Function ()

    用於標籤切換時的事件回調函數，[ default: null ]

API

iframeTab.iframeHeight(updateHeight)

刷新iframe窗口高度。在某些場合上如通過JS對頁面進行動態修改，頁面高度改變會出現窗口高度過短，此時可通過此方法刷新頁面高度。

若在子窗口調用，需加上 parent。

parent.iframeTab.iframeHeight(updateHeight)

  • updateHeight | number

    某些情況，頁面包含脫離文檔流的元素且超出document高度，此時應獲取該元素高度并作為參數傳入。
    忽略此參數，表示取iframe的document高度。



iframeTab.closeActIframe()

執行此函數，將關閉當前iframe窗口。

按鈕

在元素上添加 [data-btn="refresh"] ，則點擊該元素時，元素所在iFrame頁將被刷新。

gulp

因為本插件涉及iframe操作，在本地直接打開將報錯，你需要在服務器上調試，本插件提供 gulp 調試方式，你可以以此進行插件模擬和代碼編譯。

  • gulp 運行服務器并編譯所有代碼
  • gulp browser-sync 運行服務器
  • gulp watch 進入watch模式
  • gulp sass-to-css 將sass編譯成css
  • gulp minify-css 壓縮css
  • gulp jscompress 壓縮js
